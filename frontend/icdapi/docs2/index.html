ICD-API Documentation

ICD API allows programmatic access to the International Classification of Diseases (ICD). It is an HTTP based REST API.

This documentation describes how to use the API and other related technology.

A more granular API Reference is available in Open API (swagger) format

ICD-API Homepage is the main entry page for accessing the ICD-API.
